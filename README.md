# TextEncryption-Decryption

A simple Python application built with Tkinter for encrypting and decrypting messages using a Vigen√®re-like cipher with a custom key.

## Features

- **Encryption**: Convert plain text into a secure encoded string using a secret key.
- **Decryption**: Decode encrypted messages back to plain text using the same secret key.
- **base64 Encoding**: Uses base64 encoding to ensure the output is safe for transmission.
- **GUI Interface**: User-friendly graphical interface built with `tkinter`.

## Requirements

- Python 3.x
- `tkinter` (usually included with Python installations)

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/vinodkantineni/TextEncryption-Decryption.git
    cd TextEncryption-Decryption
    ```

2.  Run the application:
    ```bash
    python encryptDecrypt.py
    ```

## Usage

1.  **Enter the Message**: Type the text you want to encrypt or decrypt.
2.  **Enter the Key**: Provide a secret key (password). This same key must be used for both encryption and decryption.
3.  **Select Mode**:
    - Choose **Encrypt** to turn text into code.
    - Choose **Decrypt** to turn code back into text.
4.  **Show Message**: Click the button to see the result.
5.  **Reset**: Clear all fields to start over.
6.  **Exit**: Close the application.
